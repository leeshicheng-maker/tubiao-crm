# SUPER_NODE_STRATEGY（超级节点策略）

## 定义
超级节点：在政府业务拓展中具备跨区市连接能力、可影响多位关键领导认知与合作节奏的核心关系节点。

## 在本项目中的作用
1. 缩短陌生区域冷启动时间
2. 降低关键人触达成本
3. 提升项目概念接受度与推进效率

## 数据化建模（v1）
对每个超级节点维护以下字段：
- node_id / name
- 覆盖区域（省/市/区）
- 可触达关键人数量（书记/区长/副区长/局长）
- 历史促成案例数（按公开与内部记录）
- 最近活跃时间
- 可信度与合作稳定性

## 节点评分（示例）
NodeScore = 0.35 * 覆盖广度 + 0.30 * 触达深度 + 0.20 * 历史转化 + 0.15 * 活跃度

## 与项目成熟度联动
最终拜访优先级 = 项目成熟度分 * 0.6 + 超级节点匹配分 * 0.4

## 合规边界
- 仅记录业务协同关系，不记录敏感隐私。
- 推断类关系必须标注来源、日期、置信度。
