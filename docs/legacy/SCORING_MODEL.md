# SCORING_MODEL（v1）

## 评分目标
对每条线索输出 0-100 的优先级分数，用于销售资源分配。

## 评分维度
1. 时效性（25%）
   - 信号发布时间越近，得分越高
2. 预算规模（20%）
   - 预算越高，得分越高
3. 政策匹配度（20%）
   - 与当前规划/重点领域匹配越强，得分越高
4. 可接触性（20%）
   - 关键人公开信息完整、组织路径清晰，得分越高
5. 竞争强度（15%）
   - 竞争越低，得分越高

## 公式
Score = 0.25*T + 0.20*B + 0.20*P + 0.20*A + 0.15*C

其中每个子项都归一化到 0-100。

## 分级
- A级：80-100（优先拜访）
- B级：60-79（持续跟进）
- C级：40-59（观察池）
- D级：0-39（低优先级）

## 质量要求
每条线索必须包含：
- 至少 1 条可验证来源链接
- 最低 confidence 标注
- 更新时间戳
